## Allen Coral Atlas

https://allencoralatlas.org/

The json is created manually, you need to register to the https://allencoralatlas.org/ and then download the area one by one,
for example Andaman Sea, Bermuda etc.

Then you need to extract the file and get the boudary.geojson file and copy and past to the Allen_Coral_Atlas.json. Make
sure you add the properties "ECOREGION" so the user knows the boundaries belong to which ecoregion.
